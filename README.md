# Daily-Wallpaper
This python script is used to generate a unique wallpaper containing the number of days left in this year and set it as the windows wallpaper
<br>
![An example image showcasing what is generated by the script]([http://url/to/img.png](https://github.com/Salman-a-gamer/Daily-Wallpaper/blob/main/Example.png))
<br>
## Steps for installation:
1. Download the file
2. Run Task Scheduler from Windows search / Press Win + R and type taskschd.msc
3. Set the trigger to daily
4. The action is : Start a program
5. Select your saved script
6. As an argument, "C:\path\to\your\script.py"
7. For start in, C:\path\to\your\folder

## Customization:
1. You can change the width and hight of the image generated inside the script based on your display settings,
       ` WIDTH, HEIGHT = 1920, 1080 `
2. You can also add / customize the display text to your liking ,
       `text = f"{days_left}" `
